# Array Transpose

## About the project:
  Array transpose is an application that swaps the rows and columns of an Array, Example:
  </br>
  Matrix :</br>
    |  3  | -2  |  5  |
    | --- | --- | --- |
    | 3 | 0 | 4 |
Transpose :</br>
    |  3  |  3  |
    | --- | --- | 
    | -2 | 0 |
    | 5 | 4 |
